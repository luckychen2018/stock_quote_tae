# A股股票实时行情获取工具

这个工具可以帮助你实时获取A股股票的最新行情数据，支持上海、深圳和北京证券交易所的股票。

## 功能特点

- 实时获取股票最新价格
- 显示涨跌幅、涨跌额等关键指标
- 显示开盘价、收盘价、最高价、最低价
- 显示成交量和成交额
- 显示买卖五档行情
- 简单易用，适合新手

## 环境要求

- Python 3.6 或更高版本
- 网络连接

## 安装步骤

1. 确保你的电脑上已经安装了Python。如果没有安装，可以从 [Python官网](https://www.python.org/) 下载并安装。

2. 安装脚本所需的依赖库：

   ```bash
   pip install -r requirements.txt
   ```

## 使用方法

1. 运行脚本：

   ```bash
   python get_stock_quote.py
   ```

2. 默认情况下，脚本会获取中远海控(601919)的实时行情数据。

3. 如果你想获取其他股票的行情数据，可以修改脚本中的 `stock_code` 变量：

   ```python
   if __name__ == "__main__":
       # 修改这里的股票代码即可获取其他股票的行情
       stock_code = "601919"  # 中远海控
       print(f"正在获取股票代码 {stock_code} 的实时行情数据...")
   ```

## 股票代码说明

- 上海证券交易所的股票代码以 `6` 开头，例如：601919（中远海控）
- 深圳证券交易所的股票代码以 `0` 或 `3` 开头，例如：000001（平安银行）
- 北京证券交易所的股票代码以 `8` 开头，例如：831550（贝特瑞）

## 注意事项

1. 这个工具使用新浪财经的API获取数据，数据更新频率取决于新浪财经的更新频率。

2. 在非交易时间（如周末、节假日、股市休市时间），获取的数据可能是前一个交易日的收盘数据。

3. 如果遇到网络问题或API变更，可能会导致数据获取失败。

## 技术原理

这个工具通过以下步骤获取股票行情数据：

1. 构建新浪财经API的请求URL
2. 发送HTTP请求获取原始数据
3. 解析返回的数据字符串
4. 计算涨跌幅、涨跌额等指标
5. 格式化输出结果

新浪财经API的URL格式为：
```
http://hq.sinajs.cn/rn=时间戳&list=股票代码
```

其中：
- `时间戳` 用于防止缓存
- `股票代码` 是带市场前缀的完整代码，如 `sh601919`（上海市场）、`sz000001`（深圳市场）

## 常见问题

### Q: 为什么获取不到数据？
A: 可能的原因包括：
- 网络连接问题
- 股票代码错误
- 新浪财经API变更
- 非交易时间

### Q: 如何获取多个股票的数据？
A: 你可以修改脚本，多次调用 `get_stock_quote()` 函数获取不同股票的数据。

### Q: 数据更新频率是多少？
A: 新浪财经的API数据更新频率大约是每秒一次，但实际更新频率可能会有所不同。
